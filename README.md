# 🎓 ProjectHelper — Telegram бот для помощи с учебными проектами

> 🤖 AI-ассистент для генерации идей проектов и помощи студентам  
> Работает через локальную модель Ollama (gemma3:12b)

**Учебный проект** • Python • Telegram Bot • SQLite • Ollama

---

## 🌟 Возможности

### ✍️ Тема по запросу
- Генерация идеи проекта на основе описания пользователя
- Помощь в формулировке и доработке темы
- Ответ строго в формате проекта

### 🤖 Сгенерированная ИИ тема
- Полностью случайная идея проекта
- Простая и реализуемая тема
- Подходит для школьников и студентов 1 курса
- Уникальность за счёт анализа предыдущих тем

### 🧠 Помощь с темой
- Доработка уже выбранной темы
- Практические шаги реализации
- Конкретные рекомендации для проекта

### ⭐ Избранные темы
- Добавление сгенерированной темы в избранное
- Просмотр списка сохранённых тем
- Удаление последней добавленной темы
- Хранение данных в базе SQLite

---

## 🤖 Как работает AI

- Используется локальный сервер **Ollama**
- Модель: `gemma3:12b`
- Запросы отправляются на:
  ```
  http://localhost:11434/api/generate
  ```
- Тайм-аут запроса: 120 секунд
- Ответ используется без стриминга
- Все ответы формируются по строгим системным промптам

Бот поддерживает три режима генерации:
- идея по запросу
- помощь
- уникальная тема

## ⚙️ Установка и запуск

### 1️⃣ Клонирование репозитория

```bash
git clone https://github.com/chasingthedreams/ProjectHelper.git
cd ProjectHelper
```

### 2️⃣ Создание файла `.env`

Создайте файл `.env` в корне проекта:

```env
TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

Получить токен можно через:
👉 https://t.me/BotFather

---

### 3️⃣ Установка зависимостей

```bash
pip install pyTelegramBotAPI requests python-dotenv
```

---

### 4️⃣ Запуск

```bash
python main.py
```

Бот запускается через:

```python
if __name__ == "__main__":
    bot.polling(none_stop=True)
```

## 🗄 База данных

Используется SQLite (`bot.db`).

Таблицы:

- `users` — информация о пользователях
- `requests` — история запросов к AI
- `favorites` — сохранённые темы

База создаётся автоматически при запуске.
